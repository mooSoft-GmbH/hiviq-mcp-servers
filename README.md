# hiviq-mcp-servers

Monorepo for MCP (Model Context Protocol) servers developed for [hiviq](https://hiviq.com).

## Structure

```
hiviq-mcp-servers/
├── servers/        # Individual MCP servers
├── packages/       # Shared utilities and libraries
└── README.md
```

## Getting Started

Each server lives in its own directory under `servers/`. See the individual server README for setup instructions.
